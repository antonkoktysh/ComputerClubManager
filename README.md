# Система Учета Компьютерного Клуба

Прототип системы для мониторинга работы компьютерного клуба. Программа читает лог событий из файла, обрабатывает их в
соответствии с правилами клуба, отслеживает занятость столов и подсчитывает итоговую выручку и время работы каждого
стола за день.

## Сборка проекта

Для сборки проекта используется **CMake**.

   ```bash
   mkdir build
   cd build
   ```

Соберите проект:

   ```bash
   cmake ..
   make
   ```

4. Исполняемый файл `computer_club` будет создан в директории `build/bin/`.

## Запуск программы

Программа запускается из командной строки, принимая в качестве первого аргумента путь к текстовому файлу с входными
данными.

```bash
# Пример запуска
./build/bin/computer_club path/to/your_test_file.txt
