# Система Учета Компьютерного Клуба

Прототип системы для мониторинга работы компьютерного клуба. Программа читает лог событий из файла, обрабатывает их в
соответствии с правилами клуба, отслеживает занятость столов и подсчитывает итоговую выручку и время работы каждого
стола за день.

## Предварительные требования

Для сборки и запуска проекта вам понадобятся:
*   **Git**
*   **CMake** (версия 3.20 или выше)
*   Компилятор C++ с поддержкой стандарта C++20

## Сборка и запуск

### 1. Клонирование репозитория

Склонируйте репозиторий на свой локальный компьютер:

```bash
git clone https://github.com/antonkoktysh/ComputerClubManager.git
cd ComputerClubManager
```

### 2. Сборка проекта

Для сборки проекта используется **CMake**. Создайте директорию для сборки и соберите проект:

```bash
mkdir build
cd build
cmake ..
make
```

Исполняемый файл `computer_club` будет создан в директории `build/bin/`.

### 3. Запуск программы

Программа запускается из командной строки, принимая в качестве первого аргумента путь к текстовому файлу с входными данными.

```bash
# Запуск из корневой директории проекта
./build/bin/computer_club path/to/your_test_file.txt
```

### 4. Пример использования

В репозитории есть папка `examples` с примерами входных файлов. Вы можете запустить программу с одним из них:

```bash
./build/bin/computer_club examples/test_input_1.txt
```

## Формат входных данных

Программа ожидает на вход текстовый файл определенного формата.

1.  **Количество столов**: Первая строка содержит целое число — количество столов в клубе.
    ```
    3
    ```
2.  **Время работы**: Вторая строка содержит время открытия и закрытия клуба в формате `ЧЧ:ММ ЧЧ:ММ`.
    ```
    09:00 19:00
    ```
3.  **Стоимость часа**: Третья строка — целое число, стоимость одного часа игры.
    ```
    100
    ```
4.  **События**: Последующие строки описывают события в хронологическом порядке. Каждая строка состоит из времени, ID события и имени клиента, а также номера стола для некоторых событий.
    ```
    <Время> <ID_События> <Имя_клиента> [<Номер_стола>]
    ```
    *   `<Время>`: `ЧЧ:ММ`
    *   `<ID_События>`:
        *   `1`: Клиент пришел.
        *   `2`: Клиент сел за стол.
        *   `3`: Клиент ожидает.
        *   `4`: Клиент ушел.
    *   `<Имя_клиента>`: Строка без пробелов.
    *   `[<Номер_стола>]`: Целое число, указывается только для события с ID `2`.

    Пример последовательности событий:
    ```
    10:30 1 client1
    10:45 2 client1 1
    11:00 1 client2
    11:30 3 client2
    12:00 4 client1
    ```

## Тестирование

Для запуска тестов используется фреймворк **Google Test**. Сначала соберите проект, а затем запустите исполняемый файл тестов:

```bash


./bin/computer_club_tests
```

## Структура проекта

*   `src/`: Исходный код основной библиотеки и исполняемого файла.
*   `include/`: Заголовочные файлы для библиотеки.
*   `tests/`: Исходный код тестов.
*   `examples/`: Примеры входных файлов для запуска программы.
*   `CMakeLists.txt`: Главный сборочный скрипт CMake.
